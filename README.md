# Журнал работ (Work Log App)

## Стек 

### Frontend
- React + TypeScript
- axios
- vite
Выбран за простоту, скорость разработки.

### Backend 
- Nest + Typescript
- Prisma ORM
- MySQL
Nest - выбран за готовую архитектуру, разделением на модули, контроллеры и сервисы, что упрощает поддержку и масштабирование проекта.
Prisma - дает типобезопасную работу с бд, простой синтаксис.

# Запуск 
1. Клонировать или скачать репозиторий.
2. В корне `docker compose up --build`

## Доступ к приложению

- Frontend: http://localhost:4173  
- Backend: http://localhost:3000
